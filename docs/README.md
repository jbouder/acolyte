# Acolyte Documentation

This directory contains the [Docusaurus](https://docusaurus.io/) site for the
Acolyte documentation, published to GitHub Pages at
<https://jbouder.github.io/acolyte/>.

## Local development

```bash
cd docs
npm install
npm start
```

Starts a local dev server with hot reload at <http://localhost:3000>.

## Build

```bash
npm run build
```

Generates static content into the `build/` directory. Preview it with:

```bash
npm run serve
```

## Structure

- `docs/` — Markdown documentation pages.
- `src/css/custom.css` — theme overrides.
- `static/img/` — images (logo, favicon, social card).
- `docusaurus.config.ts` — site configuration.
- `sidebars.ts` — sidebar (autogenerated from the folder structure).

## Deployment

Deployment is automated via
[`.github/workflows/docs.yml`](../.github/workflows/docs.yml): pushes to `main`
that change anything under `docs/` build and deploy the site to GitHub Pages.
Pull requests that touch `docs/` build the site as a check.
