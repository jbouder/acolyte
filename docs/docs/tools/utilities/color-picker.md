# Color Picker

**Route:** `/color-picker`

Pick a color and convert it between formats.

## Features

- **Multiple color spaces** — HEX, RGB, HSL, and HSV.
- **Live conversion** between formats as you adjust the color.
- **Copy values** for use in CSS or design tools.

## Using it

1. Open **Color Picker** from the sidebar.
2. Choose a color visually or enter a value in any supported format.
3. Copy the value in the format you need.
