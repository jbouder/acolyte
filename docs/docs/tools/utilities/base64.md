# Base64 Encoder / Decoder

**Route:** `/base64`

Convert text and files to and from [Base64](https://developer.mozilla.org/en-US/docs/Glossary/Base64)
encoding.

## Features

- **Encode and decode** in both directions.
- **Text and file** input support.
- **Copy to clipboard** for quick reuse.

## Using it

1. Open **Base64 Encoding** from the sidebar.
2. Choose **Encode** or **Decode**.
3. Enter text (or upload a file) and copy the result.

## Notes

- Conversion happens entirely in your browser — nothing is uploaded.
