# Image Tools

**Route:** `/image-tools`

Crop, resize, convert, and generate favicon assets from an uploaded image — all in
the browser.

## Features

- **Crop** to a selection or aspect ratio.
- **Resize** to target dimensions.
- **Convert** between formats — PNG, JPEG, and WebP.
- **Favicon generation** — produce favicon assets from a source image.

## Using it

1. Open **Image Tools** from the sidebar.
2. Upload an image.
3. Crop, resize, or convert it, then download the result. For favicons, generate
   and export the asset set.

## Notes

- Image processing happens entirely in your browser — your image is never
  uploaded to a server.
