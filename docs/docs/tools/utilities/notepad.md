# Notepad

**Route:** `/notepad`

A persistent scratch pad for quick notes, with Markdown support.

## Features

- **Markdown-aware** editing.
- **Auto-persisted** to your browser so notes survive reloads.
- **Always a click away** for jotting down snippets and reminders.

## How it works

Notepad content is stored locally in your browser (IndexedDB), so it stays
available between sessions on the same device. Nothing is uploaded.

## Using it

1. Open **Notepad** from the sidebar.
2. Start typing — your content is saved automatically.
