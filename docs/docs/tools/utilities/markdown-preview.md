# Markdown Preview

**Route:** `/markdown-preview`

A live Markdown editor with side-by-side preview.

## Features

- **Real-time rendering** as you type.
- **GitHub Flavored Markdown** (tables, task lists, strikethrough) via
  [`remark-gfm`](https://github.com/remarkjs/remark-gfm).
- **Syntax highlighting** for fenced code blocks.
- **Export** your rendered content.

## Using it

1. Open **Markdown Preview** from the sidebar.
2. Type or paste Markdown in the editor.
3. See the rendered output update live alongside it.
