# Games

**Route:** `/games`

A small collection of browser-based games for when you need a break.

## Included games

| Game | Route | Description |
| --- | --- | --- |
| **Snake** | `/games/snake` | The classic arcade game with modern controls and scoring. |
| **Breakout** | `/games/breakout` | Brick-breaker with physics-based ball movement. |
| **Sudoku** | `/games/sudoku` | Number puzzle with multiple difficulty levels. |

## Using them

1. Open **Games** from the sidebar.
2. Pick a game and play. Everything runs locally in your browser.
